ctestrunner
===========

Playing around with C and Node in C Test Runner.

 - Requires NodeJS
 
###Usage:
 
 Run `nodejs test_runner.js <YOURFILE.c> dependencies`
 
 To run the example run:
 
     nodejs test_runner.js test.c add

>It :
>	 ✔ Adds 0+0
>	 ✔ Adds 2+2
>	 ✔ Adds 2+ -52
>	 ✔ Says 1+1 = 2
>	 ✔ Works with ok
